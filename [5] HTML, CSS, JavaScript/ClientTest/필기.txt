HTML
- Hyper Text Markup Language
	- Hyper Text: 링크
	- Markup Language: 표기 형태
- SGML > 표현에 관련된 기능들만 발추 + 모아놓은 언어 + 사용법 쉬움

Markup Language
- 마크업 표기법(<>)을 사용해서 정의된 언어
- SGML > 메타 언어(언어를 만들어낼 수 있는 언어) > HTML, XML

HTML 1.0 > HTML 2.0 > .. > HTML 4.01(***) > XHTML 1.0 > HTML5

HTML 개발 환경

- IDE > 이클립스, Visual Studio

HTML 실행 환경
- 브라우저(실행)


HTML 실행하는 방법
1. 로컬 실행(X)
	- 파일을 직접 브라우저를 통해서 연결 + 실행
2. 서버 실행(*****)
	- 파일을 서버를 통해서 브라우저에게 전달 + 실행
